Returns the next node in the queue, in container form. Contains array of elements and element count. Also contains total vertice floats, indices and indices size. Once the node data has been returned, it is discarded.

Returns
Bulk Render Container with element array and amount of elements, as well as total vertice floats, indices and indices size.

Note: Getting the size of the array from the object itself will NOT return the number of elements.
