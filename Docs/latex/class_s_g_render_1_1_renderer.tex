\doxysubsection{Detailed Description}
First you generate the renderer, and add instances if you want instanced rendering When you use the renderer, you submit\+:
\begin{DoxyEnumerate}
\item The vertices and indices you wish to batch together
\item Then any additional instance variables for drawing the batched geometry multiple times
\end{DoxyEnumerate}

You then buffer video data and draw primitives
