\doxysubsection{Detailed Description}
Container to manage all objects in a scene -\/ will improve ID allocation in future Contains render and update groups, as well as a render and update heap for ungrouped components Heap components and game objects must be std\+::shared\+\_\+ptr objects Also contains mutexes for different data to ensure threaded capabilities
